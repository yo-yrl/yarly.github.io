204. Count Primes

Posted on 2020-04-27

题目

Count the number of prime numbers less than a non-negative number, n.

Example:

Input: 10
Output: 4
Explanation: There are 4 prime numbers less than 10, they are 2, 3, 5, 7.

题意分析

给定一个数，找出这个数范围的 质数 的个数。

解题思路

  • 对于质数来说，是 只能对1或其本身除尽。
  • 可以对每个数进行判断是否是质数，然后进行统计。
  • 对于每个质数（2，3，5，等），我们可以排除他的倍数不是质数。比如 2 可以排除 4,6,8,10等，则3是质数，又可以排除 6,9,12等；接下来4不是质数，5是，以此循环。

202. Happy Number

Posted on 2020-04-27

题目

Write an algorithm to determine if a number n is “happy”.

A happy number is a number defined by the following process: Starting with any positive integer, replace the number by the sum of the squares of its digits, and repeat the process until the number equals 1 (where it will stay), or it loops endlessly in a cycle which does not include 1. Those numbers for which this process ends in 1 are happy numbers.

Return True if n is a happy number, and False if not.

Example: 

Input: 19
Output: true
Explanation: 
12 + 92 = 82
82 + 22 = 68
62 + 82 = 100
12 + 02 + 02 = 1

题意分析

给出了happy number的定义。给定一个数，判断他是否是“开心数”。
注意其有一个特点：如果该数不是开心数，在判断过程当中一定会陷入某种循环。

解题思路

  • 只能老老实实判断了。用set记录曾经遇到过得中间值，如果出现相同，即陷入循环。

200. Number of Islands

Posted on 2020-04-27

题目

Given a 2d grid map of ‘1’s (land) and ‘0’s (water), count the number of islands. An island is surrounded by water and is formed by connecting adjacent lands horizontally or vertically. You may assume all four edges of the grid are all surrounded by water.

Example 1:

Input:
11110
11010
11000
00000

Output: 1

题意分析

给定一个由字符 ‘0’ 和 ‘1’组成的二维数组，0代表水，1代表陆地，求给定数组里头小岛有几个。小岛的定义是：四面被水包围的陆地，设边界处都是水。

解题思路

  • 深搜思路。遍历数组，当遇到一块陆地（字符’1’）时，则将该小岛标记起来（遍历每一个相邻的’1’），表示已经访问过该小岛。然后再继续寻找下一个小岛。时间：$O(n)$

198. House Robber

Posted on 2020-04-26

题目

You are a professional robber planning to rob houses along a street. Each house has a certain amount of money stashed, the only constraint stopping you from robbing each of them is that adjacent houses have security system connected and it will automatically contact the police if two adjacent houses were broken into on the same night.

Given a list of non-negative integers representing the amount of money of each house, determine the maximum amount of money you can rob tonight without alerting the police.

Example 1:

Input: [1,2,3,1]
Output: 4
Explanation: Rob house 1 (money = 1) and then rob house 3 (money = 3).
         Total amount you can rob = 1 + 3 = 4.

题意分析

有一个盗贼，想要入屋盗窃。现小区内有一排房子，房子都装有报警器，如果偷了两家相邻的房子，则会自动报警。假设小偷知道每家每户的可盗取金额，问小偷最多能偷多少钱。

解题思路

  • 动态规划方法。 对于当前i房子，他可以选择偷或不偷，如果偷， 他不能偷 i-1家的，下一次得从i-2家投起；如果不偷，则可以再从i-1家偷起。所以可以退出一个递推式子： $dp[i] = max(dp[i-2]+money[i], dp[i-1])$.

191. Number of 1 Bits

Posted on 2020-04-26

题目

Write a function that takes an unsigned integer and return the number of ‘1’ bits it has (also known as the Hamming weight).

Example 1:

Input: 00000000000000000000000000001011
Output: 3
Explanation: The input binary string 00000000000000000000000000001011 has a total of three '1' bits.

题意分析

给定一个无符号32位整数，求取其bit位中1的个数。

解题思路

  • 确定是32位无符号整数了，我们可以逐一判断每一位。将输入n和0x1进行与操作判断最后一位是0还是1，然后将n右移一位。或者。对0x1进行向左移位，n不变。
  • 存在一个技巧：$n & n-1$会去掉n的最后一个为1的比特位。
    比如 4： 10000100， 3为： 00000011，则4&3 = 10000000。

179. Largest Number

Posted on 2020-04-25

题目

Given a list of non negative integers, arrange them such that they form the largest number.
Example 1:

Input: [10,2]
Output: "210"

Example 2:

Input: [3,30,34,5,9]
Output: "9534330"

Note: The result may be very large, so you need to return a string instead of an integer.

题意分析

给定一组整数，求他们组合起来得到的最大的数，返回其字符串。

解题思路

  • 这道题要求得到的是所有数组组合起来得到的最大的值，我们可以求取所有排列组合方式，然后最后进行比较。不过这样时间复杂度很高。$O(n!)$
  • 本题求解的是数组中的数值按照某种排序进行组合，以期得到最大的组合数。所以我们就要寻思着找出这样一种排序方案。从样例出发：如例1，两个数之间的组合 10， 2，可以组合成 210 和 102，二者进行比较，则210最大，则返回 210. 对于两个数是这样，如果多个数呢？这就类似一种两两比较的排序问题了。比如 例2中，我们能够很容易最大，9肯定是排在其他数值前得到的值最大的，其次是5，然后是34，接下来是3，最后是30. 对于 34，3，30中的两两组合，我们可以得到 343 和 334 的组合 或 330 和 303的组合，是不是 34 排在前头比较好，然后 3 排在 30前头比较好。故而得到最大的数是 34330。

189. Rotate Array

Posted on 2020-04-25

题目

Given an array, rotate the array to the right by k steps, where k is non-negative.

Example 1:

Input: [1,2,3,4,5,6,7] and k = 3
Output: [5,6,7,1,2,3,4]
Explanation:
rotate 1 steps to the right: [7,1,2,3,4,5,6]
rotate 2 steps to the right: [6,7,1,2,3,4,5]
rotate 3 steps to the right: [5,6,7,1,2,3,4]

Note:

  • Try to come up as many solutions as you can, there are at least 3 different ways to solve this problem.
  • Could you do it in-place with O(1) extra space?

    题意分析

旋转数组：给定一个数组和非负整数k，k表示从右边数有几位数旋转到数组前头来。

解题思路

  • 一个一个进行旋转。每次保留第n个数，然后将前头n-1个数都往后挪一位，最后将第n个数放到第0位，如此循环k次。时间复杂度是$O(n*k)$,超时了。
  • 三次旋转。将数组进行倒序，然后针对前k个进行倒序，以及后n-k个数进行倒序。比如：123456, k = 2, 第一步：654321， 第二步：56 4321， 第三步：56 1234. 时间复杂度：$O(n)$

169. Majority Element

Posted on 2020-04-25

题目

Given an array of size n, find the majority element. The majority element is the element that appears more than ⌊ n/2 ⌋ times.

You may assume that the array is non-empty and the majority element always exist in the array.

Example 1:

Input: [3,2,3]
Output: 3

Example 2:

Input: [2,2,1,1,1,2,2]
Output: 2

题意分析

寻找中位数的变形——寻找出现次数超过数组长度一半的数。

解题思路

  • 用map存储 value 和 出现次数 的映射，返回出现此处超过一半的值。$空间：O(n), 时间：O(n log(n))$ —每次map查找是log(n)
  • 由于该数值出现次数超过一半，故而他的出现次数比其他数值出现次数的总和都多。如果我们记录当前一个值及其次数，若遇到与其不一样的，次数减一。如果次数为0，则更新值及其次数。最后存储的值一定是出现次数最多的值，即返回结果。$时间：O(n), 空间：O(1)$
