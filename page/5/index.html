139. Word Break

Posted on 2020-04-14

题目

Given a non-empty string s and a dictionary wordDict containing a list of non-empty words, determine if s can be segmented into a space-separated sequence of one or more dictionary words.

Note:
The same word in the dictionary may be reused multiple times in the segmentation.
You may assume the dictionary does not contain duplicate words.

Example 1:

Input: s = "leetcode", wordDict = ["leet", "code"]
Output: true
Explanation: Return true because "leetcode" can be segmented as "leet code".

Example 2:

Input: s = "applepenapple", wordDict = ["apple", "pen"]
Output: true
Explanation: Return true because "applepenapple" can be segmented as "apple pen apple". *Note that* you are allowed to reuse a dictionary word.

题意分析

  • 题目给定一个非空字符串，和一个字符串数组，问字符串是否能根据数组内的小字符串进行分割~
  • 对于一个字符串s, 从 i = 0 位置开始，如果 s(i, j) 这个子串在数组内，则问题可以看做判断子串 s(j, end) 是否能进行分割的一个问题，我们可以考虑用深搜的方法进行求解

    解题思路

  • 用 set 记录数组内的小字符串，便于查找
  • 深搜求解，记录一个字符串的当前起始位置 sidx ，然后遍历 sidx – s.size()的位置 j , 当子串s.substr(sidx, j-sidx+1) 在 set 内， 则可以 对子问题进行求解。如果子问题返回 true ， 代表该字符串可以求解。
  • 深搜求法会出现很多重复计算的步骤，则我们可以一个数组来记录 当前开始位置为 sidx 的子字符串是否可以进行分割。
  • 动态规划方法。对于结束位置为i的字符串，dp[i]判断其是否可以分割————遍历0–i-1 位置 j , 如果 dp[j] == true （即可以分割）且 s.substr(j, i-j) 在 set 内， 则代表该字符串可以分割 –> dp[i] = true;

138. Copy List with Random Pointer

Posted on 2020-04-12

题目

A linked list is given such that each node contains an additional random pointer which could point to any node in the list or null.

Return a deep copy of the list.

The Linked List is represented in the input/output as a list of n nodes. Each node is represented as a pair of [val, random_index] where:

  • val: an integer representing Node.val
  • random_index: the index of the node (range from 0 to n-1) where random pointer points to, or null if it does not point to any node.

题意分析

给定一个链表，除了 next 指针，还有一个 random 指针，指向链表中其他节点或 nullptr 。我们要对这个链表进行深拷贝，并返回新链表。

因为新旧链表的地址不一样，我们不能通过简单的 p == p1 寻找新链表节点的 random 指针。所以我们需要想办法去解决这个新旧节点地址不同的问题。

解题思路

  • 用map进行新旧节点的映射。这样在寻找random指针时就很方便~ 遍历一遍链表，创建新链表并完成map映射，再进行一遍遍历，完成random指针的指向~ 【时间复杂度:O(n), 空间复杂度: O(n)】
  • 在原有链表上增加新节点：old1 -> new1 -> old2 -> new2，然后 新节点的Random指针就是旧节点的random指针的next；最后，将链表拆开成新旧链表，返回新链表【时间复杂度：O(n), 空间复杂度：O(1)】

136. Single Number

Posted on 2020-04-11

题目

Given a non-empty array of integers, every element appears twice except for one. Find that single one.

Note:

Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?

Example 1:

Input: [2,2,1]
Output: 1

Example 2:

Input: [4,1,2,1,2]
Output: 4

题意分析

给定一个非空整数数组，其中只有一个数字只出现一次，其他数字都出现两次，先要求找出该只出现一次的数字。

解题思路

  • 要求o(n)复杂度，一般想到就是空间换时间。可以通过记录整数出现次数来找出答案，用map来存储 value-count 数对，两次遍历解决：第一次遍历数组，记录 value-count， 第二次，则判断整数的count是否为1。
  • 利用异或解决，a ^ a = 0, 将数组每个值进行异或，则最后的值肯定是 只出现一次的那个数字。

实现代码

  • 异或方法

134. Gas Station

Posted on 2020-04-10

题目

There are N gas stations along a circular route, where the amount of gas at station i is gas[i].

You have a car with an unlimited gas tank and it costs cost[i] of gas to travel from station i to its next station (i+1). You begin the journey with an empty tank at one of the gas stations.

Return the starting gas station’s index if you can travel around the circuit once in the clockwise direction, otherwise return -1.

Note:

If there exists a solution, it is guaranteed to be unique.
Both input arrays are non-empty and have the same length.
Each element in the input arrays is a non-negative integer.

Example 1:

Input: 
gas  = [1,2,3,4,5]
cost = [3,4,5,1,2]

Output: 3

Explanation:
Start at station 3 (index 3) and fill up with 4 unit of gas. Your tank = 0 + 4 = 4
Travel to station 4. Your tank = 4 - 1 + 5 = 8
Travel to station 0. Your tank = 8 - 2 + 1 = 7
Travel to station 1. Your tank = 7 - 3 + 2 = 6
Travel to station 2. Your tank = 6 - 4 + 3 = 5
Travel to station 3. The cost is 5. Your gas is just enough to travel back to station 3.
Therefore, return 3 as the starting index.

题意分析

给定两个数组，gas和cost，gas[i]代表当前加油站的油量（全都给车加上），cost[i]表示从加油站i到i+1之间好消耗多少油量。问，能否从某个油站出发 走完所有加油站。

解题思路

  • 很直观的一个想法是进行双重遍历。尝试从每个加油站出发，然后计算是否能遍历一遍。当前剩余油量(remainGas) = 前次剩余油量 + gas[i] - cost[i], 如果 remainGas < 0， 则代表不能才能够这个加油站出发。
  • 第二种方法是：从头到尾只遍历一次，也记录 remainGas，如果 remainGas < 0, 则表示要到达改点，还需 remainGas 这么多油量（欠下的 needGas += remainGas）。然后设 remainGas = 0， 从下一个点开始走起。 遍历完成后，看remainGas是否能补足之前 欠下的 油量 needGas。

131. Palindrome Partitioning

Posted on 2020-04-09

题目

Given a string s, partition s such that every substring of the partition is a palindrome.

Return all possible palindrome partitioning of s.

Example:

Input: "aab"
Output:
[
    ["aa","b"],
    ["a","a","b"]
]

题意分析

  • 给定一个字符串，进行不同回文子串的分割，得到所有分割的组合

解题思路

  • 我们需要一个判断字符串是否是回文串的函数

  • 对于一个字符串“aab”,当前子串”a”是回文串，則可以以当前位置作为分割点，然后求剩下子串”ab”的组合。或者看另一个分割点，”aa”这是一个回文串，则这个也是一个分割点，求剩下子串“b”的组合。

  • 这是一个递归过程：对于当前子串，我们需要得到以0位置为开头的回文串，然后分割，继续递归求所有组合。

  • 我们可以记录一个start_idx记录当前子串的起始位置，递归结束条件是start_idx == input_string.size(),即代表求到当前分割方法的所有子串组合了。

128. Longest Consecutive Sequence

Posted on 2020-04-08

题目

Given an unsorted array of integers, find the length of the longest consecutive elements sequence.

Your algorithm should run in O(n) complexity.

Example:

Input: [100, 4, 200, 1, 3, 2]
Output: 4
Explanation: The longest consecutive elements sequence is [1, 2, 3, 4]. Therefore its length is 4.

题意分析

1 对于给定的无序的一组整数，找出最长连续序列的最长长度。时间要求复杂度是o(n).

2 一般要求了时间复杂度的，都会以空间换时间。
所以我们要想用啥数据结构来存一下这些数据 嘻嘻。
这个类似一个排序题，如果用排序算法，时间复杂度大于o(n);(如果是计数排序，不能确定最小值和最大值，可能会浪费大量空间)

解题思路

1 可以用set或hash_set来存储整数值（去重）。

2 然后对每个存在于数组中的整数，如果其存在set中，则对其左右进行扩散查找，得到他的连续序列长度；同时，从set中删除已经遍历过的整数值，避免重复查找。

124. Binary Tree Maximum Path Sum

Posted on 2020-04-05

题目

Given a non-empty binary tree, find the maximum path sum.

For this problem, a path is defined as any sequence of nodes from some starting node to any node in the tree along the parent-child connections. The path must contain at least one node and does not need to go through the root.

Example 1:

Input: [1,2,3]

   1
  / \
 2   3

Output: 6

题意分析

题目要求我们找出一条路径，从任意某个点开始到任意一个点结束的一条路径，使得这条路径上值的和最大。比如上述例子，是2-1-3这条路径（或者反过来）。

解题思路

一般遇到二叉树-求路径的题都会考虑遍历（递归形式），然后基于某种遍历方式，添加要额外的功能，比如该题的求一条路径的上的和。

因为要求的是和最大的路径，则对于某个点(val)来说，他当前的最大路径和等于 val+left_val+right_val的值，他返回给上一层几点的值为val + max(left_val, right_val)。

考虑后序遍历形式（从叶节点向上遍历，这样只需遍历一次就可以求得最大路径）。我们可以求得当前节点（val）的左右子树的路径最大值, 分别为left_val, right_val（如果为nullptr或值<0，则为0）。然后更新路径最大和，res = max(res, val+left_val+right_val)，返回 val + max(left, right)——返回和最大的路径。

121. Best Time to Buy and Sell Stock

Posted on 2020-04-04

题目

Say you have an array for which the ith element is the price of a given stock on day i.

If you were only permitted to complete at most one transaction (i.e., buy one and sell one share of the stock), design an algorithm to find the maximum profit.

Note that you cannot sell a stock before you buy one.

Example 1:

Input: [7,1,5,3,6,4]
Output: 5
Explanation: Buy on day 2 (price = 1) and sell on day 5 (price = 6), profit = 6-1 = 5.
            Not 7-1 = 6, as selling price needs to be larger than buying price.

题意分析

给出一个数组，表示一段时间内该股票的价格，让我们求能挣到的最多钱（即差价）。如果给出的是递减数组，则返回0（没钱可挣，别买股票了）。

解题思路

  1. 暴力破解：$O(n^2)$

    • 双重循环，求每对价格的的差价，记录最大差价。

  2. 遍历一遍：$O(n)$

    • 目标是相求最大差价：对于第i天来说，其最大差价等于当前价格 减去 在他之前的最小价格~
    • 可以用一个值来保存遍历过的数的最小值，然后对 最大差价 和 最小值进行更新
