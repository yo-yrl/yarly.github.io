283. Move Zeroes

Posted on 2020-05-07

题目

Given an array nums, write a function to move all 0’s to the end of it while maintaining the relative order of the non-zero elements.

Example:

Input: [0,1,0,3,12]
Output: [1,3,12,0,0]

Note:

  • You must do this in-place without making a copy of the array.
  • Minimize the total number of operations.

题意分析

给定一个数组，要求我们不能增加额外的空间，将数组中 的 0 值交换到数组最后，并且保持非0数组间的相对顺序（类似稳定排序的概念）。

解题思路

  • 两个指针。看到需要在原数组上对两类值进行交换位置，就考虑到了用双指针：一个指针a用于指向一类值的位置，另一个b指向另一类值的位置，由此交换两个位置的值，并且更新指针指向位置，循环致结束。
  • 保持相对顺序。一开始，就想着双指针分别从两头开始寻找对应位置，然后交换值。但是结果显示，这样做会破坏各个元素的相对位置。比如 0,1,0,3,12 的结果会是 12, 1, 3, 0, 0. 这样是不对的。所以，我们得注意替换过程中不能改变相对位置。最后，可以将连个指针都从 0 位置开始寻找，这样交换时，肯定能保存非0数值的相对位置： 比如，0,1,0,3,12, 第一次：1,0,0,3,12,第二次：1,3,0,0,12,第三次：1,3,12,0,0.
