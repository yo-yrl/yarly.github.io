131. Palindrome Partitioning

Posted on 2020-04-09

题目

Given a string s, partition s such that every substring of the partition is a palindrome.

Return all possible palindrome partitioning of s.

Example:

Input: "aab"
Output:
[
    ["aa","b"],
    ["a","a","b"]
]

题意分析

  • 给定一个字符串，进行不同回文子串的分割，得到所有分割的组合

解题思路

  • 我们需要一个判断字符串是否是回文串的函数

  • 对于一个字符串“aab”,当前子串”a”是回文串，則可以以当前位置作为分割点，然后求剩下子串”ab”的组合。或者看另一个分割点，”aa”这是一个回文串，则这个也是一个分割点，求剩下子串“b”的组合。

  • 这是一个递归过程：对于当前子串，我们需要得到以0位置为开头的回文串，然后分割，继续递归求所有组合。

  • 我们可以记录一个start_idx记录当前子串的起始位置，递归结束条件是start_idx == input_string.size(),即代表求到当前分割方法的所有子串组合了。
