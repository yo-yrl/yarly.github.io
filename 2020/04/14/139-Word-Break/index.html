139. Word Break

Posted on 2020-04-14

题目

Given a non-empty string s and a dictionary wordDict containing a list of non-empty words, determine if s can be segmented into a space-separated sequence of one or more dictionary words.

Note:
The same word in the dictionary may be reused multiple times in the segmentation.
You may assume the dictionary does not contain duplicate words.

Example 1:

Input: s = "leetcode", wordDict = ["leet", "code"]
Output: true
Explanation: Return true because "leetcode" can be segmented as "leet code".

Example 2:

Input: s = "applepenapple", wordDict = ["apple", "pen"]
Output: true
Explanation: Return true because "applepenapple" can be segmented as "apple pen apple". *Note that* you are allowed to reuse a dictionary word.

题意分析

  • 题目给定一个非空字符串，和一个字符串数组，问字符串是否能根据数组内的小字符串进行分割~
  • 对于一个字符串s, 从 i = 0 位置开始，如果 s(i, j) 这个子串在数组内，则问题可以看做判断子串 s(j, end) 是否能进行分割的一个问题，我们可以考虑用深搜的方法进行求解

    解题思路

  • 用 set 记录数组内的小字符串，便于查找
  • 深搜求解，记录一个字符串的当前起始位置 sidx ，然后遍历 sidx – s.size()的位置 j , 当子串s.substr(sidx, j-sidx+1) 在 set 内， 则可以 对子问题进行求解。如果子问题返回 true ， 代表该字符串可以求解。
  • 深搜求法会出现很多重复计算的步骤，则我们可以一个数组来记录 当前开始位置为 sidx 的子字符串是否可以进行分割。
  • 动态规划方法。对于结束位置为i的字符串，dp[i]判断其是否可以分割————遍历0–i-1 位置 j , 如果 dp[j] == true （即可以分割）且 s.substr(j, i-j) 在 set 内， 则代表该字符串可以分割 –> dp[i] = true;
