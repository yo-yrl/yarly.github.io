139. Word Break II

Posted on 2020-04-15

题目

Given a non-empty string s and a dictionary wordDict containing a list of non-empty words, add spaces in s to construct a sentence where each word is a valid dictionary word. Return all such possible sentences.

Note:

The same word in the dictionary may be reused multiple times in the segmentation.
You may assume the dictionary does not contain duplicate words.

Example 1:

Input:
s = "catsanddog"
wordDict = ["cat", "cats", "and", "sand", "dog"]
Output:
[
"cats and dog",
"cat sand dog"
]

题意分析

是139题的延伸，题目给定一个非空字符串，和一个字符串数组，求 字符串根据数组内的小字符串进行分割 而得到的所有组合。

解题思路

  • 求组合排列一般采用深搜（回溯）法。用 vector 保存每次切割的子字符串，对于字符串s，从i位置切割，如果 s(0-i) 在数组内，则 存入vector中，问题变成球子串 s(i-end) 的组合。存在多种组合，在对于下个位置的分割前，我们需要将上次 s(0-i) 的子串弹出，即从同一个起始位置开始 的 分割位置发生改变。实现代码为wordDFS()函数，遇到
    “aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaabaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa”
    [“a”,”aa”,”aaa”,”aaaa”,”aaaaa”,”aaaaaa”,”aaaaaaa”,”aaaaaaaa”,”aaaaaaaaa”,”aaaaaaaaaa”]
    输入会超时。因为aaa—aaa存在太多种组合了，耗时。
  • 使用空间，记录当前位置i是否可以进行分割，避免重复计算以及大量不必要的递归。参考139题的动态规划方法，得到 dp[] 数组。然后在 DFS 函数中根据dp判断，则可ac，不过时间还是很慢QwQ.实现代码为wordDFSDP()。
