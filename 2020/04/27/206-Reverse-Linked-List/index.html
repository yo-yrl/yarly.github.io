206. Reverse Linked List

Posted on 2020-04-27

题目

Reverse a singly linked list.

Example:

Input: 1->2->3->4->5->NULL
Output: 5->4->3->2->1->NULL

Follow up:

A linked list can be reversed either iteratively or recursively. Could you implement both?

题意分析

给定一个链表，实现链表翻转功能。

解题思路

  • 用栈存储节点信息。遍历链表，用栈存储每个节点的值。再遍历一遍，将链表的值替换成栈顶的值，并出栈。
  • 三指针方法。用三个指针 *pre, *cur, *next 分别记录当前遍历节点，及其前后两个节点，便于将指针翻转过来。翻转过程： 比如 1->2->3; pre = 1, cur = 2, next = 3; cur->next = pre; 1<-2->3; 然后使得pre = cur = 2; cur = next = 3, next = nullptr; 以此循环。
  • 递归方法。
